# aimeet.github.io
#ABOUT
AI MEET is a desktop application that helps the user to record audio of online meetings, later it transcribe the audio and summarized it with key points and action items for boosting productivity. 

#REQUIRMENTS
-> Windows 10/11 (x64 Bit)
-> Stable internet connection
